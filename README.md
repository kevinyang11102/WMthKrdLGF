# 酒店管理系统 java14

## 项目概述

酒店管理系统是一款基于B/S架构和MVC模式的系统，采用前后端分离的开发模式，主要使用Java和Vue技术栈进行构建。系统支持用户登录、注册、预约住房等多种功能，同时为管理员提供全面的管理模块。

## 技术栈

- 架构: B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：
  - IDEA
  - JDK1.8
  - Maven
  - Mysql
  - Node.js 14
  - Navicat

技术框架：

- Java
- Mysql
- SpringBoot
- Mybatis
- Vue

## 功能模块

### 用户模块

- 用户登录
- 用户注册
- 预约住房
- 留言
- 查看公告
- 查看预约
- 入住查询

### 管理员模块

- 管理员管理
- 用户管理
- 房型管理
- 房间管理
- 预约管理
- 入住管理
- 留言管理
- 公告管理

## 系统角色

系统主要分为用户和管理员两种角色。

- 用户：使用系统进行登录、注册、预约住房等操作。
- 管理员：负责管理系统，包括用户、房型、房间、预约、公告等方面的管理。

## 运行环境

- 操作系统：Windows/Mac
- 数据库：Mysql
- 服务器：Tomcat（或其他支持Java的应用服务器）
- 前端环境：Node.js

## 部署步骤

1. 安装并配置后端开发环境（IDEA、JDK1.8、Maven等）
2. 导入数据库文件并创建数据库
3. 编译并运行后端项目
4. 安装并配置前端开发环境（Node.js、npm等）
5. 编译并运行前端项目
